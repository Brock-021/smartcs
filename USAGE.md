## 用户端 — 智能客服

### 访问地址

浏览器打开 `http://your-server:8080/` (或通过 Nginx 代理的域名)

### 基本使用

1. **发起咨询** — 在聊天框输入问题，AI 自动回答
2. **发送图片** — 点击 📎 图标选择图片
3. **转人工** — 输入"转人工"或点击 ❓ 转人工按钮
4. **评价服务** — 工单关闭后，可星级评分

### 用户注册/登录

> 需要管理员在后台开启用户注册功能

1. 点击页面顶部「登录」按钮
2. 输入邮箱和密码登录，或点击「注册」创建账号
3. 登录后可查看历史工单记录
4. 再次点击头像可退出登录

### PWA 使用 (移动端)

- **iOS Safari** — 分享 → 添加到主屏幕
- **Android Chrome** — 菜单 → 添加到主屏幕
- 安装后可离线访问，推送通知

## 客服端 — 工作台

### 登录

访问 `/agent/login`，使用客服账号登录。

### 工单列表

左侧工单列表分为三个标签页：

| 标签 | 说明 |
|------|------|
| ❇️ 待处理 | 新的未分配工单 |
| 👤 我的客户 | 已分配给当前客服的工单 |
| 📋 历史工单 | 已完成的工单 |

### 工单处理流程

```
待处理 → 受理(抢占) → 沟通回复 → 请求关闭 → 用户确认 → 已关闭
                                             ↓ 用户超时
                                           自动关闭
```

1. **接单** — 点击"受理"按钮，工单进入"处理中"状态
2. **回复** — 输入消息后发送，客户实时收到
3. **请求关闭** — 问题解决后，点击「请求关闭」选择原因
4. **重开** — 用户未确认时，可重新打开

### 辅助功能

- **升级工单** — 将工单升级给更高级别的客服
- **提交缺陷** — 关联 Jira/禅道，直接创建缺陷单
- **查看客户信息** — 点击客户名称查看详情

### 快捷键

- `Enter` — 发送消息
- `Shift+Enter` — 换行

## 管理端 — 后台管理

### 登录

使用管理员账号 (admin@smartcs.com) 在客服登录页登录，然后点击右上角「后台管理」。

### 功能标签

| 标签 | 功能 |
|------|------|
| 📋 **工单管理** | 查看/搜索全部工单，导出 CSV |
| 📊 **数据统计** | 工单趋势、客服排行、满意度统计 |
| 👥 **用户管理** | 管理注册用户（增删改查） |
| 👤 **客服管理** | 管理客服账号（增删改查、级别设置） |
| ⚙️ **系统配置** | AI API 密钥、自动关闭时间等 |
| 🏷️ **关闭原因** | 自定义工单关闭原因标签 |
| 📋 **系统升级** | 记录系统升级日志 |
| 🔧 **负责系统** | 配置服务目录/系统列表 |
| 🔗 **集成网关** | 管理外部系统对接 (企微/钉钉/Jira/禅道) |
| 🔐 **认证管理** | LDAP/OIDC 认证提供商配置 |
| 📋 **审计日志** | 全量操作审计记录 |
| 🎨 **品牌配置** | 品牌名称、主题色、Logo 后台可配（v4.1 新增） |
| 🔒 **安全配置** | 密码策略、登录限制、审计保留配置（v4.1 新增） |

### 关键操作说明

#### 客服管理
1. 进入「客服管理」标签
2. 点击「添加客服」，填写邮箱、姓名、密码
3. 可设置客服级别 (1-5，高级可管理低级)
4. 可启用/禁用客服账号

#### 系统配置
- **AI 模型** — 切换 AI 引擎 (DeepSeek/通义千问)
- **知识库** — 配置知识库路径
- **自动关闭** — 设置解决后自动关闭的等待时间(分钟)
- **自动评价** — 设置用户未确认时的自动评价时间(小时)

#### 🎨 品牌配置
1. 进入「品牌配置」标签页
2. 修改品牌名称、品牌简称、主题色（hex 色值，如 `#1a73e8`）
3. 配置 Logo 和 Favicon 图片路径
4. 点击「保存配置」，修改即时生效
5. 所有页面标题、颜色、Logo 自动更新

#### 🔒 安全配置
1. 进入「安全配置」标签页（安全管理员专属）
2. 设置密码最小长度（建议 ≥ 8）
3. 开启/关闭大写字母要求
4. 设置密码过期天数（建议 90 天）
5. 配置登录失败锁定策略（尝试次数、锁定分钟数）
6. 配置审计日志保留天数
7. 点击「保存配置」

#### 集成网关配置

**企业微信：**
1. 进入「集成网关」→ 添加适配器 → 选择「企业微信」
2. 填写 CorpID、AgentID、CorpSecret
3. 启用后即可向企微用户推送消息

**钉钉：**
1. 添加适配器 → 选择「钉钉」
2. 填写 AppKey、AppSecret
3. 启用后可通过钉钉发送通知

**Jira/禅道：**
1. 添加适配器 → 选择「缺陷系统」
2. 配置 API 地址、认证信息
3. 客服可在工单中直接提交缺陷

#### 知识库管理

访问 `/upload` 页面：
1. 输入管理员密码验证
2. 拖拽或点击上传知识文档
3. 支持格式：Word (.docx)、PDF、PPT、Markdown、TXT
4. 可在线预览/编辑已有文档
5. AI 自动学习知识库内容用于回复

## 系统集成配置

### LDAP 认证

1. 后台「认证管理」→ 添加认证提供商 → LDAP
2. 填写:
   - 服务器地址 (ldap://your-ldap-server:389)
   - Base DN (dc=example,dc=com)
   - 绑定 DN 和密码
3. 启用后，客服登录页显示 LDAP 登录按钮

### OIDC (SSO)

1. 后台「认证管理」→ 添加认证提供商 → OIDC
2. 填写:
   - Issuer URL (https://keycloak.example.com/realms/xxx)
   - Client ID / Client Secret
   - 回调地址
3. 启用后，客服登录页显示 SSO 登录按钮

### Webhook 通知

1. 后台「集成网关」→ 添加 Webhook
2. 配置回调 URL 和签名密钥
3. 选择触发事件 (工单创建、状态变更、转人工等)
4. 系统自动推送 JSON 格式通知到指定 URL
