# SmartCS 变更日志

## v4.1 (2026-05-21)

### 🚀 新增功能
- **品牌自定义** — 后台可配置系统名称、品牌简称、主题色、Logo、Favicon
- **安全策略管理** — 后台可视化配置密码策略、登录限制、审计保留天数
- **CSS 变量化** — 全站主题色跟随品牌配置，一处修改全局生效
- **PWA 动态化** — manifest.json、service-worker 动态读取品牌配置

### ⚙️ 系统优化
- 所有硬编码品牌名称/颜色 → 后台可配置
- 会话超时、空闲超时、分页大小、上传限制 → 配置化管理
- 6 个模板 + app.py 全部改为动态配置读取

### 📝 文档
- 新增 SmartCS-产品手册.md
- 更新 README / USAGE / architecture / 需求文档 / 设计文档
- 更新测试计划与测试报告

## v4.0 (2026-05-20)

### 🚀 新增功能
- **用户注册/登录系统** — 聊天页面顶部登录入口，支持注册和密码登录
- **历史工单查询** — 登录用户可查看自己的工单记录和处理状态
- **全量审计日志** — 记录所有客服/管理员的操作，后台可查看和筛选
- **PWA 支持** — 添加到桌面，离线缓存
- **移动端底部导航** — 聊天/工单/设置 Tab 切换

### 🐛 问题修复
- 客服登录无反应 (cookie 路径问题)
- 工单列表 SQL 错误 (ambiguous column)
- 消息送达但界面不显示 (模板缓存问题)
- 历史聊天记录为空 (状态机阻塞)
- 图片显示为纯文字 (Blob URL 格式)
- 客户名为系统默认名 (visitor → 客户)
