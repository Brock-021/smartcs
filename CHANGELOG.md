# SmartCS 变更日志

## v4.0 (2026-05-20)

### 🚀 新增功能
- **用户注册/登录系统** — 聊天页面顶部登录入口，支持注册和密码登录
- **历史工单查询** — 登录用户可查看自己的工单记录和处理状态
- **全量审计日志** — 记录所有客服/管理员的操作，后台可查看和筛选
- **PWA 支持** — 添加到桌面，离线缓存
- **移动端底部导航** — 聊天/工单/设置 Tab 切换

### 🐛 问题修复
- 客服登录无反应 (cookie 路径问题)
- 工单列表 SQL 错误 (ambiguous column)
- 消息送达但界面不显示 (模板缓存问题)
- 历史聊天记录为空 (状态机阻塞)
- 图片显示为纯文字 (Blob URL 格式)
- 客户名为系统默认名 (visitor → 客户)
